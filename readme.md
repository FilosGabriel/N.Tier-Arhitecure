# N.Tier Architecture

## Technologies

- Language: Java 17
- Base Framework: Spring Boot 2.5.7
- Web:
    - Spring Web (Servlet)
- Databases:
    - PostgresSQL with Spring Data JPA(classic version)
    - Liquidbase - migrations PostgresSQL
- Testing:
    - Spock
    - Spring testing
- Utility:
    - Mapstruct
    - Lombok
    - Spring Validations
